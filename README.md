# Topogram Operations Design Review Proof

> A focused proof that designers and front-end leads can review a substantial Topogram UI work map from Markdown packets before reading raw `.tg`.

This repo demonstrates a Vite React operations dashboard using a bespoke Acme Operations UI component system. Later checkpoints add a Topogram work map, design language, component map, designer packet, and slice-guided UI change.

Run the canonical proof command at every checkpoint:

```sh
npm ci
npm run verify
```

The React app is inspectable with:

```sh
npm run dev:react
```

See [proof/README.md](proof/README.md) for the step map.

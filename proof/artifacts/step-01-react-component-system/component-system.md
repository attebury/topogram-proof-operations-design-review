# Step 01 Component System

The baseline app is a Vite React operations dashboard using a bespoke Acme Operations UI component system.

## Stable Component Refs

- `acme.dataGrid`
- `acme.filterRail`
- `acme.commandBar`
- `acme.alertBanner`
- `acme.timeline`
- `acme.statusBoard`
- `acme.detailPanel`
- `acme.workloadHints`

## What To Inspect

- Intake Queue: dense filters, command bar, data grid, and workload hints.
- Incident Detail: detail panel and timeline.
- Assignment Board: status board/cards.
- Dashboard: KPI cards and alert banner.

There is intentionally no `topo/` folder in this checkpoint.
